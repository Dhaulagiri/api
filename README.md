# Versioned API Browser
This Ember Application allows you to browse API docs

# Setup

1. Install Dependencies

  ```shell
  npm install && bower install
  ```

1. Generate Documentation JSON

  ```shell
  mkdir -p tasks/docs
  node tasks/generate-docs.js
  ```

1. Start the app

  ```shell
  ember server
  ```

1. View the app
  
  ```shell
  open http://0.0.0.0:4200
  ```
